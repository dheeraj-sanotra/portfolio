Zerava Chat

Real-Time WebSocket Chat Application

laravel/zerava-websocket-chat

Production-ready real-time chat application built with Laravel Reverb, React 19, and TypeScript. Covers one-to-one messaging, group conversations, typing indicators, read receipts, file attachments, presence tracking, and push notifications.

Architecture

Design Patterns & Decisions

Laravel Reverb for first-party WebSocket broadcasting — no Pusher dependency

Presence channels for real-time online / offline / away status

Service + Repository architecture on the backend; component-driven React frontend

Queue workers for attachment processing, image optimisation, and notification delivery

Policy-based authorisation on every message, conversation, and group action

4-service runtime: API, Reverb WebSocket, Queue worker, React dev server

Features

What's Built

Messaging

  • One-to-one direct messages and public / private group conversations
  • Real-time delivery via WebSocket broadcasting with typing indicators and read receipts
  • Message reactions, emoji support, and delivery status tracking
  • Full-text search across messages, users, and groups

Presence & Profiles

  • Live online / offline / away status with last-seen timestamps
  • Active users list (24 h window) and per-conversation presence
  • Avatar upload and user profile management

Attachments & Notifications

  • Image and document uploads with background processing and secure local storage
  • Real-time broadcast notifications + database persistence + email delivery
  • Optional Firebase FCM push notifications for mobile / background alerts

Admin

  • Admin dashboard for user management, account suspension, and activity monitoring
  • Analytics on active users, message volume, and group activity
